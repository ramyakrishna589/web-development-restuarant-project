# 🍽️ Restaurant Website - Web Development Project

This is a fully responsive, modern restaurant website built using **HTML**, **CSS**, and **JavaScript**. It showcases a food business's offerings with an elegant UI, smooth interactions, and responsive design across devices.

---

## 🚀 Features

- ✅ Home, Menu, About, and Contact sections
- ✅ Responsive design for mobile, tablet, and desktop
- ✅ Animated scrolling and transitions
- ✅ Stylish navigation bar with smooth scrolling
- ✅ Call-to-action buttons
- ✅ Footer with social media links

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (Vanilla JS)
- Google Fonts & Font Awesome
